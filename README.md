
# Social-Network

Project for social network by NodeJs and ReactJs


## Build Project

1.Build server

```bash
  cd api
  yarn install
  yarn start
  #do not forget to create .env before yarn start
  #in file .env, add endpoint of MongoDB
  #MONGO_URL="endpoint of MongoDB"
```
2.Build client

```bash
  cd client
  yarn install
  yarn start
  #Run on http://localhost:3000/
```
